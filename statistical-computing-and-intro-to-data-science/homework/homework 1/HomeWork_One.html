The cor() function calculates the correlation coefficient between two variables. What is the correlation between property logValue and property logUnits in (i) the whole data, (ii) just Manhattan (iii) just Brooklyn (iv) for properties built after 2000 (v) for properties built before 2000? You will need to add the argument use = “pairwise.complete.obs” to handle NA values.

# the whole data
whole <- housing %>% select(logValue,logUnits) %>% summarise(correlation_of_whole = cor(logValue,logUnits,use = "pairwise.complete.obs"))
